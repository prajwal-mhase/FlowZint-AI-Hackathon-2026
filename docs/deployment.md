# Deployment Guide

Deployment will be finalized in Phase 7.

## Intended Targets

- Frontend: Vercel
- Backend: Railway or Render
- Database: managed PostgreSQL
- Cache: managed Redis
- Vector database: hosted Chroma-compatible service or containerized Chroma
- File storage: Supabase Storage

## Local Infrastructure

The initial local infrastructure file is available at `infra/docker-compose.yml`.
